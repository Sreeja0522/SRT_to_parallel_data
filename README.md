In all series_input the srt files for each episode are named as 1.srt , 1.srt and placed inside series_named folders.

The code recognises the language whether english/telugu by the file content and number in the filename.

The paths can be checked inside main.py for input folder(All_Series_Input_original)

First main.py needs to be executed and outputs can be checked for any mistakes.

The timestamps for the spilt lines can be found in Process_Internal_stages which is output of main.py.

Then main2.py needs to be executed to recieve the final parallel data in separate folders.

The documentataion for all the codes can be found in documentation of codes.pdf
